what is github ?

GitHub is a developer platform that allows developers to create, store, manage and share their code. It uses Git software, providing the distributed version control of Git plus access control, bug tracking, software feature requests, task management, continuous integration, and wikis for every project.

Steps to send data on github repository

Step 1 : create an account on github.com

Step 2 : download git software to send data on github via git bash

Step 3 : git init

Step 4 : git add .

Step 5 : git commit -m '25-09-2024 all data updated'

Step 6 : git remote add origin https://github.com/Brijesh1990/reactjs-mwf-6to7-30pm.git
